# Three.js + React で波紋のアニメーションを実装する

## まずはプロジェクトを用意

```bash
npx create-react-app [プロジェクト名]
```

## ReactでThree.jsを扱うためのライブラリをインストール

```bash
npm install three react-three-fiber
```
